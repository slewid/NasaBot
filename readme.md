# NasaBot Slack Bot

A Slack bot that retrieves NASA's **Astronomy Picture of the Day (APOD)** using the NASA APOD API and posts it into Slack.

## Features

- Fetch today's Astronomy Picture of the Day
- Fetch a random APOD
- View HD images when available
- Support for video APODs

---

| Command | Description |
|---------|-------------|
| `/nasabot-apod` | Get today's Astronomy Picture of the Day |
| `/nasabot-hd-apod` | Get today's APOD in HD |
| `/nasabot-random` | Get a random APOD |
| `/nasabot-hd-random` | Get a random APOD in HD |
| `/nasabot-ping` | Check if the bot is online |
| `/nasabot-help` | Display all available commands |
